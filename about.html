Tapwave Zodiac Game Install Process

For the longest time we all used to use HotSync, a program similar to iTunes to set up and install files to the Zodiac. Honestly, today's method is actually a bit easier.

Many apps are packaged as just a .prc and possibly a .pdb file as well. Just copy these to /PALM/Launcher on your SD card and you're good to go.

If an app has a required game data folder there will also be a folder in the download. It will have a name like Zod-fcps. That folder needs to be copied to /PALM/Programs on your SD card.

The name of the data folder is displayed for your convenience.

PopAccess

You need to have PopAccess on your Zodiac to run most games.

Mac OSX

Sometimes, especially in Zodiac apps with file browsers, you'll see some junk files that start with a period. Mac OSX puts these files all over the card for reasons we don't care about. Download Blueharvest to clean them.
